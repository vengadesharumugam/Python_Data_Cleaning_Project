# Python Data Cleaning & Simple Linear Regression

## Project Overview

This project demonstrates core data cleaning steps and a simple linear regression model using Python.  

It covers:

- Handling missing values

- Detecting and removing outliers (IQR method)

- Building a simple linear regression model to predict Purchase Amount from Annual Income

## Steps Performed

1. Loaded raw customer dataset

2. Checked and handled missing values

3. Detected outliers using IQR and removed extreme values

4. Trained a simple linear regression model

5. Evaluated the model using R² score

6. Exported cleaned datasets and prediction outputs

## Files

- `data/` – input CSV files used for cleaning and modeling

- `notebooks/data_cleaning_project.ipynb` – main notebook with all code

- `outputs/` – cleaned data and model predictions

- `README.md` – project description
